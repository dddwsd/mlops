# Redis Example with python

## Install and run redis server
```bash
# Mac OS
$ brew install redis
# Linux
$ apt-get install redis-server
```

## Set environment
```bash
$ python3 -m venv ~/.venv/redis
$ source ~/.venv/redis/bin/activate
$ pip insatll -r requirements.txt
```

## Run example code
```bash
python3 redis_client.py
```
